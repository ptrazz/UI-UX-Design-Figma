# UI-UX-Design-Figma
Memory Game (Flashcard)

This is a UI/UX design prototype for an English–Indonesian flashcard learning application created in Figma. The interface is designed with game-inspired elements to make learning more engaging and enjoyable. The prototype includes a landing page, login and sign-up pages, a main menu, and game screens. Users can learn new vocabulary through interactive flashcards and review their progress with quizzes. A scoring system tracks the number of correct answers, providing immediate feedback and encouraging continuous learning.
